# tintinland-class3

### 脚本
```
# 安装npm包
npm install -g truffle
truffle version
npm install @truffle/hdwallet-provider --save
npm install @openzeppelin/contracts --save

# truffle初始化
truffle unbox

# 部署
truffle migrate --network development

# 单元测试
truffle test .\test\ERC20Token.test.js --network development
```
